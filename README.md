# 🌉 تانل خلیفه - Khalifeh Tunnel

مدیریت تونل Rathole برای اتصال امن و پایدار بین سرور ایران و خارج

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/xperee/khalifeh-tunnel)

## ✨ ویژگی‌ها

- 🔐 پیکربندی آسان در دو طرف (ایران و خارج)
- 🌐 پشتیبانی کامل از IPv4 و IPv6
- 🔄 تونل‌زنی همزمان TCP و UDP
- ⚙️ مدیریت سرویس‌ها با systemd
- 🚀 بهینه‌سازی خودکار شبکه (BBR, TCP_NODELAY)
- 📊 مشاهده لاگ و وضعیت سرویس‌ها
- 🔄 قابلیت بروزرسانی خودکار
- 🎯 مناسب برای استفاده با 3x-ui و Xray

## 📋 پیش‌نیازها

- سرور با سیستمعامل لینوکس (Ubuntu/Debian/CentOS)
- دسترسی روت (root)
- اتصال به اینترنت

## 🚀 نصب سریع

```bash
bash <(curl -s https://raw.githubusercontent.com/xperess/Khalifeh_Tunnel_v3/main/install.sh)
📖 راهنما
مرحله 1: نصب روی هر دو سرور
دستور نصب را روی هر دو سرور ایران و خارج اجرا کنید.

مرحله 2: پیکربندی سرور ایران
گزینه 1 را انتخاب کنید

تنظیمات زیر را وارد کنید:

پورت تونل (مثلاً 20000)

توکن امنیتی (می‌تواند تصادفی تولید شود)

پورت‌هایی که می‌خواهید تونل شوند (مثل 443, 80, 8080)

مرحله 3: پیکربندی سرور خارج
گزینه 2 را انتخاب کنید

اطلاعات دریافتی از سرور ایران را وارد کنید:

IP سرور ایران

پورت تونل

توکن امنیتی

پورت‌های محلی

مرحله 4: بهینه‌سازی شبکه (اختیاری)
گزینه 6 را انتخاب کنید تا تنظیمات شبکه بهینه شود.

🛠 دستورات مدیریتی
دستور	توضیحات
khalifeh	اجرای منوی اصلی
systemctl status khalifeh-*	مشاهده وضعیت سرویس‌ها
journalctl -u khalifeh-iran-20000 -f	مشاهده لاگ زنده
systemctl restart khalifeh-iran-20000	راه‌اندازی مجدد سرویس
🔄 سناریوی استفاده با 3x-ui
text
کاربر → سرور ایران → تونل خلیفه → سرور خارج → 3x-ui → اینترنت
روی سرور خارج: 3x-ui را نصب و کانفیگ کنید

روی سرور ایران: تونل خلیفه را به عنوان سرور تنظیم کنید

روی سرور خارج: تونل خلیفه را به عنوان کلاینت تنظیم کنید

در کانفیگ 3x-ui، از IP سرور ایران استفاده کنید

📁 ساختار فایل‌ها
text
/root/khalifeh-tunnel/
├── rathole                 # فایل اجرایی Rathole
├── iran-*.toml            # کانفیگ‌های سمت ایران

/etc/systemd/system/
├── khalifeh-iran-*.service   # سرویس‌های سمت ایران
└── khalifeh-kharej-*.service # سرویس‌های سمت خارج
